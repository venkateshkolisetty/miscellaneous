=head1 PROPERTIES

=over

=item 'collapsed' (boolean : default false : readable / writable / private)

Whether the group has been collapsed and items are hidden

=item 'ellipsize' (Pango::EllipsizeMode : default "none" : readable / writable / private)

Ellipsize for item group headers

=item 'header-relief' (Gtk2::ReliefStyle : default "normal" : readable / writable / private)

Relief of the group header button

=item 'label' (string : default "" : readable / writable / private)

The human-readable title of this item group

=item 'label-widget' (Gtk2::Widget : default undef : readable / writable / private)

A widget to display in place of the usual label

=back



=cut


=head1 CHILD PROPERTIES

=over

=item 'expand' (boolean : default false : readable / writable / private)

Whether the item should receive extra space when the group grows

=item 'fill' (boolean : default true : readable / writable / private)

Whether the item should fill the available space

=item 'homogeneous' (boolean : default true : readable / writable / private)

Whether the item should be the same size as other homogeneous items

=item 'new-row' (boolean : default false : readable / writable / private)

Whether the item should start a new row

=item 'position' (integer : default 0 : readable / writable / private)

Position of the item within this group

=back



=cut


=head1 STYLE PROPERTIES

=over

=item 'expander-size' (integer : default 16 : readable / private)

Size of the expander arrow

=item 'header-spacing' (integer : default 2 : readable / private)

Spacing between expander arrow and caption

=back



=cut
