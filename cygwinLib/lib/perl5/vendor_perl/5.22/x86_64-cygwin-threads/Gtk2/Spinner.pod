=head1 PROPERTIES

=over

=item 'active' (boolean : default false : readable / writable)

Whether the spinner is active

=back



=cut


=head1 STYLE PROPERTIES

=over

=item 'cycle-duration' (Glib::UInt : default 1000 : readable)

The length of time in milliseconds for the spinner to complete a full loop

=item 'num-steps' (Glib::UInt : default 12 : readable)

The number of steps for the spinner to complete a full loop. The animation will complete a full cycle in one second by default (see #GtkSpinner:cycle-duration).

=back
